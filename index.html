Privacybeleid

Versie 1.1 · Ingangsdatum: maart 2026

Deze app is gemaakt door een onafhankelijke Bredase ontwikkelaar, uit liefde voor de Bredase horeca.

Welke gegevens slaan we op?

De app slaat uitsluitend lokaal op jouw apparaat op: beoordelingen, notities, lijstjes en jaarboekgegevens. Deze verlaten jouw telefoon niet.

Google Places API

De app gebruikt Google Places API voor restaurantinfo. Geen persoonlijke gegevens worden gedeeld met Google.

Geen account, geen tracking

Geen loginscherm, geen advertenties, geen tracking.
